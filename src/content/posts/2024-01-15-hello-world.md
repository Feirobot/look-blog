# 你好，世界！👋

经过几天的开发和调试，我的个人博客终于正式上线了！

## 为什么建这个博客

在这个社交媒体和短视频盛行的时代，为什么还要坚持写博客呢？

1. **深度思考**：博客让我能够系统地整理和记录自己的想法
2. **知识沉淀**：写下来的知识才是真正掌握的知识
3. **分享交流**：希望能通过文章帮助到有同样困惑的人
4. **个人品牌**：建立自己的技术影响力

## 技术栈

这个博客使用以下技术构建：

- **Astro 4.x** - 现代静态站点生成器，零 JavaScript 输出
- **Tailwind CSS** - 原子化 CSS 框架，快速开发
- **TypeScript** - 类型安全，更好的开发体验
- **Markdown** - 简洁的内容创作方式

选择 Astro 的主要原因是它出色的性能和简洁的开发体验。默认零 JavaScript 的输出让页面加载速度非常快，Lighthouse 评分轻松达到 95+。

## 未来计划
